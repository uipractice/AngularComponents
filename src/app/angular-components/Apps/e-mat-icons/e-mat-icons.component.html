Description

Angular supports Material icons. If you want to use material icons in your project then you need to download svg icons first then you need to import 'MatIconRegistry' from '@angular/material' and it's builtin service so need to inject within constructor with serviceName after that need to write like this in constructor body
serviceName.addSvgIcon('name of the Icon','Icon path need to register Here').

Description

Get vector icons and social logos on your website with Font Awesome, the web's most popular icon set and toolkit.

Setup

Bootstrap 4 dropped Glyphicons support, So we have to use some other icon font, like Font Awesome.

Step 1: Run below command in your project terminal.

npm i bootstrap

Step 2: Place respective bootsrap.min.css, bootstrap.min.js and jQuery files path in angular.json file.

Step 3: Add "https://use.fontawesome.com/releases/v5.0.8/js/all.js" in index.html
